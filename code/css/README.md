# CSS

Linting and style checking playbooks for CSS, SCSS, Sass, and Less stylesheets.

## Playbooks

| Playbook | Tool | Description |
|----------|------|-------------|
| [lint/stylelint.yml](lint/stylelint.yml) | Stylelint | Mighty, modern linter that helps avoid errors and enforce conventions in CSS, SCSS, Sass, and Less styles |

## Usage

```bash
satori run ./ --playbook satori://code/css/lint/stylelint.yml --report --output
```
